PAQUETE LISTO PARA RENDER

1. Abre esta carpeta localmente.
2. Pega tu código dentro de main.py (reemplaza el contenido).
3. Sube todo a un repositorio de GitHub.
4. En Render:
   - New → Worker
   - Build Command: pip install -r requirements.txt
   - Start Command: python main.py
5. Deploy.

Listo, tu bot correrá 24/7.
